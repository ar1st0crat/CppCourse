Detailed Description

Represents the Card entity.

Card numbers : 0..51

  • 0..12 -> hearts suit = 0
  • 13..25 -> diamonds suit = 1
  • 26..38 -> clubs suit = 2
  • 39-51 -> spades suit = 3

Card ranks:

2..10 J=11 Q=12 K=13 A=14 ( J, Q, K, A will be interpreted differently depending on the game type )
